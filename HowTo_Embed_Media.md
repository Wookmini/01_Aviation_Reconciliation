# 🖼️ 이미지 및 동영상 추가 가이드

이 문서는 `Automation_Case_Study.md` 파일(웹페이지)에 이미지나 동영상을 넣는 방법을 잊지 않도록 기록해 둔 가이드입니다. 

문서에 삽입된 모든 미디어 파일은 배포된 웹사이트에서 **자동으로 둥근 모서리, 부드러운 그림자 등의 고급스러운 글래스모피즘(Glassmorphism) 스타일이 적용**되도록 세팅되어 있습니다.

---

## 1. 이미지 넣기 (Markdown 문법)

가장 기본적인 마크다운 문법을 사용하여 이미지를 삽입할 수 있습니다. 이미지는 가급적 이 프로젝트 폴더 내에(예: `images/` 폴더 안) 함께 두고 경로를 지정하는 것이 좋습니다.

```markdown
![여기에 이미지 설명 적기](이미지_파일_경로.png)
```

**✅ 사용 예시:**
```markdown
![성공적인 정산 화면](images/success_screenshot.jpg)
```
*(주의: 파일 경로에 띄어쓰기가 있다면 동작하지 않을 수 있으니, 파일명은 언더바(`_`)나 하이픈(`-`)을 사용하는 것이 좋습니다.)*

---

## 2. 동영상 넣기 (HTML 문법)

화면 녹화 파일(mp4 등)을 넣고 싶을 때는 HTML의 `<video>` 태그를 그대로 작성하시면 됩니다. 마크다운 변환기가 이를 인식하여 웹페이지에 비디오 플레이어를 띄워줍니다.

### 기본 재생 플레이어 (재생바 컨트롤 포함)
사용자가 직접 재생 버튼을 눌러야 재생되는 형태입니다.
```html
<video src="동영상_파일_경로.mp4" controls></video>
```

### 🔁 움짤(GIF)처럼 자동 재생 + 반복 재생하기
버튼을 누르지 않아도 소리 없이 계속해서 무한 반복 재생되는 형태입니다. (시연 영상 등을 넣을 때 가장 추천합니다!)
```html
<video src="동영상_파일_경로.mp4" controls autoplay loop muted></video>
```

**✅ 사용 예시:**
```html
<video src="media/automation_demo.mp4" controls autoplay loop muted></video>
```
