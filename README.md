# 항공권 정산 자동화 시스템 (Aviation Reconciliation Automation)

이 프로젝트는 e-DAHS 리스트와 세중 리스트 간의 항공권 정산 내역을 자동으로 비교하고 검증하여 업무 효율성을 높이는 도구입니다.

## 주요 기능
- **데이터 정규화**: 서로 다른 형식의 엑셀 데이터를 파싱하고 비교 가능한 형태로 정규화합니다.
- **자동 대조**: 두 리스트 간의 불일치 항목(금액 차이, 누락 항목 등)을 신속하게 추출합니다.
- **결과 리포트**: 대조 결과를 엑셀 파일로 생성하여 실무자가 즉시 확인할 수 있도록 지원합니다.

## 프로젝트 구조
- `final_reconciliation.py`: 메인 정산 및 대조 로직 스크립트
- `analyze_excel.py`: 엑셀 파일 구조 분석 및 전처리 유틸리티
- `check_rows.py`: 데이터 행 수 및 기본 무결성 검사
- `excel_workspace/`: 정산에 필요한 데이터 파일이 저장되는 작업 디렉토리

## 시작하기

### 사전 요구 사항
- Python 3.x

### 의존성 설치
```bash
pip install pandas openpyxl
```

### 실행 방법
1. `excel_workspace/` 폴더 내에 정산할 엑셀 파일들을 배치합니다.
2. 메인 스크립트를 실행합니다.
```bash
python final_reconciliation.py
```

## 데이터 관련 유의사항
보안 및 용량 문제로 실제 항공권 데이터 파일(`.xls`, `.xlsx`)은 저장소에서 제외되었습니다. 사용자는 `excel_workspace/` 폴더 내에 본인의 데이터를 정해진 형식에 맞춰 배치한 후 사용해야 합니다.
